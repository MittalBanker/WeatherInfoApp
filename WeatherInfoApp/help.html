This Application has three main modules


HOME

You can switch trough Map and listing screens through icons. In Map screen you can add map pin by long press. In Listing you can see the locations which you added. You can delete them by swipe gesture.


CITY

You can see temperature, humidity etc here.


SETTINGS

In settings screen you can change the unit by which you can see the weather data and you can reset the cities which you bookmarked.
